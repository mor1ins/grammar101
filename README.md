# gramm101

Этот проект создан в качестве аналога [regex101](regex101.com), но для грамматик.
В данный момент доступен только [Ohm](https://github.com/harc/ohm), но
мы планируем это исправить. 

Проект написан на React.


## Deployment

```shell
npm install
npm start
```
